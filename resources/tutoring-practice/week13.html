Solutions

Default Parameters

Conceptual Questions:

  1. A default parameter in Python functions is a parameter that assumes a default value if no value is provided for it during the function call.

def power(base, exponent=2):
    return base ** exponent

print(power(3))  # By default
print(power(3, 3))  # Now you have overriden the default value

  1. Since x does not have a default value, directly calling the function without parameters will raise an Error.

  2. In Python function definitions, non-default parameters must precede default parameters. As otherwise will create errors for 1. positional argument assignment; 2. letting Python identify which parameter(s) are optional.
